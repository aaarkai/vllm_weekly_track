# Weekly Release Report for vllm-project/vllm (2026-06-12)

This week merged 263 PRs from 149 contributors. Key areas: features 5, fixes 70, performance 23.

## Executive Summary

本周更新引入多项模型支持与性能改进。模型层面新增 DiffusionGemma 支持，并扩展了 MiniCPMV 批预处理与 DeepSeek-V4 的 XPU attention 解码路径。性能方面，针对 AMD RDNA3 添加了 Fused MoE W4A16 HIP 内核，并进行了 MLA chunked-context 张量固定、ROCm 下 fused softplus-sqrt-topk 路由器及 flash-decode split-K 解码等优化。Marlin 内核迁移至 torch stable ABI 为后续维护做准备。

修复工作覆盖广泛，包括 Llama4、Qwen3.5、Cohere2MoE、Gemma4、Nemotron 等多个模型的权重加载与推理精度问题，以及工具内容处理、FlashInfer 条件限制、AOT 编译缓存冲突、KV cache offloading 死锁与内存泄漏。MoE 重构（#41184）引发了多项权重加载回归，已针对性修复。无重大 breaking change，但建议升级用户注意 MoE 相关配置及模型权重加载的兼容性。
