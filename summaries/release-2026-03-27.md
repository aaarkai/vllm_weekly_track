# Weekly Release Report for vllm-project/vllm (2026-03-27)

This week merged 243 PRs from 120 contributors. Key areas: features 2, fixes 51, performance 15.

## Executive Summary

本周发布的主要更新围绕功能增强、错误修复及性能优化展开。新增了支持批处理的 `/v1/chat/completions/batch` 聊天补全端点，并针对 ROCm 平台完善了硬件信息收集功能。错误修复是本期的重点，涉及范围广泛，包括修复了 AudioFlamingo3 与 Hugging Face 的 parity 问题、Qwen3.5 FP8 在 Blackwell 上的精度下降、多节点 allreduce fusion 以及多处与 ROCm、MoE、CPU 后端、CUDA Graphs 和 LoRA 相关的缺陷。

在性能方面，对 SM120 的 FP8 GEMM 内核进行了优化，并为 CPU 后端默认启用了 tcmalloc 以提升效率。模型支持持续扩展，新增了对 Granite 4.0 1B speech 的支持，并改进了多模态模型的处理逻辑。需要注意的是，本次发布已弃用 `pooling multi task` 功能，这构成了一项破坏性变更，建议用户在升级前评估影响。
