# Weekly Release Report for vllm-project/vllm (2026-04-17)

This week merged 205 PRs from 139 contributors. Key areas: features 9, fixes 56, performance 14.

## Executive Summary

本周发布主要聚焦于核心性能优化与模型生态扩展。关键特性包括引入TurboQuant以实现2位KV缓存压缩，将容量提升4倍；CPU后端现已支持草案模型推测解码；基础设施方面升级至transformers v5，并对`requirements/`目录进行了结构化调整。新增了对EXAONE-4.5、Jina Embeddings v5以及Gemma4 Eagle3等模型的支持，同时XPU平台增加了FP8 KV缓存支持。

缺陷修复涵盖了优先级归一化、稀疏注意力、多模态处理、多专家模型及多种硬件后端（如ROCm、XPU）相关的多项问题，提升了系统稳定性。性能优化涉及延迟导入以加速CLI启动、针对Blackwell架构的专家混合层配置调优以及减少主机到设备的内存拷贝。本次升级需注意transformers v5可能引入的变更，建议测试后部署。
