# Weekly Release Report for vllm-project/vllm (2026-05-15)

This week merged 254 PRs from 149 contributors. Key areas: features 6, fixes 58, performance 19.

## Executive Summary

本周版本围绕多硬件平台适配、模型兼容性修复与推理性能优化展开。功能方面，新增Blackwell架构下DSR1/Kimi K25的TOKENSPEED_MLA预填充与解码后端、RISC-V向量扩展注意力内核及NVFP4全规约GEMM融合异步张量并行。修复涵盖DeepSeek V4系列多项数值与状态错误、Gemma 4工具解析器流式浮点损坏、EXAONE-4.5对齐Transformers更新、Qwen3-VL流水线并行初始化等关键缺陷。性能提升包括MLA预填充内存分配优化、W8W8组量化二维网格消除取模、Cutlass FP8批量不变性实现28.9%端到端延迟改进，以及消除注意力实现中GPU-CPU同步。无明确破坏性变更，但需注意实验性HTTP路由专家替换以及RVV内核、TOKENSPEED_MLA等新功能需验证对应硬件兼容性。升级建议：确认DeepSeek V4相关修复涉及cutlass与Triton内核参数调整，建议用户同步更新依赖至最新提交。
