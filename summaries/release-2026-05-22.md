# Weekly Release Report for vllm-project/vllm (2026-05-22)

This week merged 218 PRs from 137 contributors. Key areas: features 3, fixes 52, performance 17.

## Executive Summary

## 本周发布摘要

本周期主要围绕架构迁移、性能优化与模型扩展展开。核心亮点包括：激活内核等组件持续迁移至 libtorch 稳定 ABI；Rust 前端代码从独立仓库迁入主仓库；强化学习场景下的 IPC 权重同步实现多 GPU 支持与分块打包张量优化；新增 OpenVLA 模型支持；Hybrid Mamba 状态后处理引入融合 Triton 内核。

功能方面，新增 cuDSS dsv4 索引器 FP8 内核、池化器激活函数单元测试，以及 Mamba SSM 缓存的 bf16 支持。大量 Bug 修复覆盖了 CI 稳定性、FlashInfer、MoE 路由、KV Offload、DeepSeek V4/V3、Cohere 视觉模型、CPU 后端、结构化输出等多个模块。性能改进涉及 Cutlass FP8 线性核预权重处理（TTFT 提升 13.5%）、MLA 解码全 CUDA Graph 捕获、GPU-CPU 同步消除及 RISC-V 注意力核首次支持等。模型侧还完成了 DeepSeek V4 模块重命名与 MiniCPM-V 权重加载修正。

未发现破坏性变更或特殊升级指引。
