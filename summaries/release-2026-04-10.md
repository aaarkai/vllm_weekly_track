# Weekly Release Report for vllm-project/vllm (2026-04-10)

This week merged 185 PRs from 119 contributors. Key areas: features 3, fixes 37, performance 13.

## Executive Summary

本周发布的版本主要聚焦于内核优化与模型支持的扩展。性能方面，通过重构 W8A8 Block Linear 操作并统一 Persistent TopK 调度器内核，提升了计算效率；同时，多项优化工作覆盖了 GEMM 管理、注意力机制与 KV 缓存交换等方面。功能上，新增了对 EXAONE-4.5 模型的支持，并在 CPU 上实现了草案模型推测解码，语法功能也得到增强。

缺陷修复涉及多个关键领域，包括修复了量化 KV 缓存、MoE 模型、Gemma4 工具调用解析以及 NUMA 绑定等问题，增强了系统稳定性。模型与硬件支持方面，扩展了对 Qwen、Phi-4 等新模型的支持，并改进了在 AMD 和 XPU 硬件上的兼容性。本次更新包含若干重大变更，例如对 NVFP4 GEMM 管理的重构、部分 V0 接口的弃用以及持续集成流程的调整，建议用户在升级前仔细查阅相关说明。
