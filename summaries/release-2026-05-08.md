# Weekly Release Report for vllm-project/vllm (2026-05-08)

This week merged 177 PRs from 123 contributors. Key areas: features 3, fixes 39, performance 11.

## Executive Summary

本周发布涵盖多项新功能、广泛Bug修复与性能改进。功能层面，vLLM Chat Completions API新增prompt_embeds支持，xgrammar更新实现严格结构化工具调用，并加入nvfp4 KV cache支持。模型支持扩展至AMD DeepSeek V4、Moondream3以及Qwen3.5/3.6的Gated DeltaNet注意力。性能优化包括消除GPU-CPU同步、集成Tile Kernels for DeepSeek-V4、Helion配置解析优化及ROCm DBO。Bug修复重点解决tensorizer LoRA OOM、DeepSeek属性解析、flashinfer autotune禁用、FP8 bias加载等问题。重要变更：RayExecutorV2已默认启用，现有用户需验证升级兼容性。
