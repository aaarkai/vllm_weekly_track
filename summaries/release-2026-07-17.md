# Weekly Release Report for vllm-project/vllm (2026-07-17)

This week merged 231 PRs from 142 contributors. Key areas: features 5, fixes 51, performance 14.

## Executive Summary

本版本重点围绕KV缓存重构、稀疏MLA优化、混合模型前缀缓存及ROCm平台改进展开。核心特性包括：定义清晰的KV Offload后端配置边界（#48150）、支持混合模型的部分前缀缓存命中（#46384）、为稀疏MLA短序列新增密集MHA路径（#47327），以及跨注意力后端的K/V打包重构（#44455）。ROCm方面引入了HybridW4A16线性核（Triton prefill + HIP skinny decode，#40977）。性能优化涵盖Qwen3.5在H20上的加速（#48350）、FP32路由器GEMV优化（#48335）、FlashInfer MNNVL allreduce与RMS量化融合（#48064），以及ROCm DSv4两阶段压缩器（#47718）。此外，新增Inkling模型及其LoRA/MTP支持、Cosmos3 Edge Reasoner模型，并实现了FlashInfer MoE LoRA BF16集成（#48632）。修复涉及池化量化、安全CVE（#48583）、FlashInfer非因果草稿注意力（#48167）、TurboQuant FP8在Ampere GPU上的失败（#39988）等数十项。模型支持还包括LlavaNextVideo LoRA、fp32 lm_head配置及DeepseekV32文档。未引入重大破坏性变更，建议更新后验证KV缓存相关配置及ROCm内核调用。
