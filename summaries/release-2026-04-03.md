# Weekly Release Report for vllm-project/vllm (2026-04-03)

This week merged 199 PRs from 127 contributors. Key areas: features 8, fixes 49, performance 16.

## Executive Summary

本次发布在模型支持、性能优化与系统稳定性方面取得多项进展。核心新增对 Google Gemma 4 架构（支持 MoE、多模态、推理及工具调用）的完整支持。性能方面，引入了通用的 CPU KV Cache Offloading 机制，并在 MLA attention 中支持 fused FP8/NVFP4 输出量化，同时为 Marlin GEMM/MoE 添加了 MXFP8 支持。

错误修复涵盖范围广泛，重点解决了多模态处理、工具解析、量化加载、缓存管理及特定硬件（如 ROCm、NVFP4 DGX Spark）下的各类问题，并修复了多个导致崩溃或挂起的缺陷。此外，发布了针对 Cheers 等多模态模型的支持，并持续进行内核迁移与性能调优工作。本次更新未包含重大破坏性变更。
