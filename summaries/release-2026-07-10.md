# Weekly Release Report for vllm-project/vllm (2026-07-10)

This week merged 269 PRs from 165 contributors. Key areas: features 6, fixes 81, performance 17.

## Executive Summary

本周发布重点包括：新增 KVConnector push 模式下的流水线并行 prefill 支持、Transformers modeling 后端性能优化至接近原生 vLLM 水平、集成 TRTLLM BF16 MoE Modular Kernel 提升推理效率，以及加入 Humming MoE 量化后端。此外，新增 Laguna XS.2.1 DFlash drafter 支持、TorchCodec 视频解码后端，并为 AMD Instinct MI355 添加 tuned selective_state_update 配置，CPU 端也补充了 Mamba ShortConv 支持。

修复方面覆盖了模型 runner、量化、分布式、MLA 解码、RoPE 填充、安全漏洞（如 unbounded engine fan-out）等多个领域，包括 MRV2 的 num_accepted_tokens 重置、KVBlockZeroer 竞态条件、FlashInfer MLA workspace 类型、Hopper MXFP4 OOB 读取、DeepSeek-V4 KV cache reshape 等。无已知重大破坏性变更，建议用户按常规流程升级，注意验证 TRTLLM 和 Humming MoE 相关配置的兼容性。
